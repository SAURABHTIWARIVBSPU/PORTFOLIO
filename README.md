# Saurabh Tiwari Portfolio

A modern personal portfolio website for Saurabh Tiwari, built with React, TypeScript, Tailwind CSS, Framer Motion, and Three.js. The site highlights frontend development experience, live projects, technical skills, education, certifications, resume download, and a WhatsApp-enabled contact form.

**Live URL:** [saurabh-tiwari.netlify.app](https://saurabh-tiwari.netlify.app)

## Features

- Responsive portfolio layout for desktop, tablet, and mobile devices
- Light neo-brutal UI theme with polished cards and section backgrounds
- 3D animated hero background using Three.js
- Optimized WebP profile images for faster loading
- Downloadable resume PDF
- Skills section with custom SVG technology icons
- Live project cards for AltFTool, Alt+F Meet, and Omegal
- Contact form that opens WhatsApp with pre-filled project details
- Production-ready Vite build setup

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS 4
- Vite
- Three.js
- Framer Motion
- Lucide React

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
